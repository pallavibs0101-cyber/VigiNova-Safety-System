# VigiNova – AI Powered Safety System

VigiNova is an AI-powered safety system designed to protect women and teenagers in real-time, even without internet.

## Key Features:
- AI-based panic detection
- 3-level emergency alert system
- Offline SMS alerts
- Live location tracking

## Prototype:
This project includes a basic prototype and system design prepared for hackathon submission.

## Goal:
To build a complete working solution during the 24-hour hackathon.
